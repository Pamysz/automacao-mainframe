# Automação Mainframe - Excel - Web Scraping

Sistema de automação Python que integra planilhas Excel, web scraping na Amazon e sistemas Mainframe.

1. **Lê produtos** de planilha Excel
2. **Pesquisa na Amazon** coletando nome, preço e descrição
3. **Insere dados** no sistema Mainframe
4. **Atualiza a planilha** com as informações obtidas

## Tecnologias
- Python • Selenium • OpenPyXL • PyAutoGUI
